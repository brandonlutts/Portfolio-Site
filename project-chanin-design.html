What I built

The design utilizes a grid block structure with stacking mobile optimization to avoid mobile corwding and overwhelming users with wall of text and endless scrolling. Short simple Clear.

  • lightweight design minimal features, clean, simple straight to business
  • Enhanced heirachy, custom mobile css
  • interactive elements, pages added

Under the hood

  • Static HTML/CSS/ behind weebly and plugin php.
  • Responsive layout tuned for phone-in-hand learning about what they have to offer and how you can help.
  • JSON-LD event markup to help search engines direct donation and those in need to this rescue mission.
  • custom edits to css ensured that new elements will follow those same guides ensuring a lasting update to mobile ressponsiveness and layout structure for years to come.
