Bluegrass Bowl Tecmo Tournament

A focused event site for a Tecmo Super Bowl tournament hosted in the Bluegrass region. The goal: make it easy for players to understand the format, register, and get excited about the event — while also showing off a bit of NES-era charm.

What I built

The site gives players everything they need to know about the tournament: format, location, rules, and how to sign up. The design leans into a Tecmo-style vibe while still being readable and mobile-friendly.

  • Landing page with a clear call-to-action for registration.
  • Sections for rules, schedule, and FAQs.
  • Space reserved for standings / bracketing and results.
  • Copy tuned to the Tecmo community: casual, direct, and hype-friendly.

Under the hood

  • Static HTML/CSS/JS for fast loads and easy hosting.
  • Responsive layout tuned for phone-in-hand checking on event day.
  • JSON-LD event markup to help search engines understand the tournament.
  • Reusable components and tokens so future years can re-use the structure.
