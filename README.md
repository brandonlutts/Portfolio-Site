# BrandonLutts.com

A custom-built portfolio experience combining cinematic UI design, motion systems, and performance-focused frontend engineering.

Inspired by HUD interfaces, architectural layouts, and modern interactive storytelling.

## Features

- Responsive custom interface
- GSAP-powered animations
- Interactive slider systems
- Scroll-triggered motion effects
- Performance optimized assets
- Modular JavaScript architecture
- Mobile-first responsive design
- SEO and metadata optimization

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GSAP
- Apache/.htaccess optimization

## Performance Work

- Minified CSS and JavaScript
- Optimized cache-control headers
- Improved Lighthouse performance scores
- Lazy-loading and asset refinement
- Mobile UX optimization
