Fairhaven Rescue Mission

A revamp and responsive restructuring for a Mens shelter/Rescue Mission in Covington, KY. They had a well established existing website that was built in wordpress with quite a few plugins. I was tasked with making their mobile site less cluttered and easy to read and scroll. Also the addition of a heart filling dontation button to make users see and feel that they filled a heart with each dontaion.

What I built

The mobile revampe utilized custom css to give the users all the nesary information without overwhelming them with wall of text and endless scrolling. Short simple Clear.

  • Enhanced heirachy, custom mobile css

Under the hood

  • Static HTML/CSS/ bedind wordpress and plugin php.
  • Responsive layout tuned for phone-in-hand learning about what they have to offer and how you can help.
  • JSON-LD event markup to help search engines direct donation and those in need to this rescue mission.
  • custom edits to css ensured that new elements will follow those same guides ensuring a lasting update to mobile ressponsiveness and layout structure for years to come.
